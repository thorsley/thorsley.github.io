Motocross app

Login

Home page of the app allowing users to login or register.

Maps

The maps page allows the user to search and easily find motocross tracks to ride.

Track

The track page is what the user will see after selecting a track. It displays all methods of contacting the track. Info will let the user see what type of vehicles are permitted on the track and what fees there are to ride. Events shows all upcoming events and ficilitates purchasing tickets for the desired event. Leaderboard shows the riders ranking on the chosen track. Forcast gives weather updates as well as track conditions.
